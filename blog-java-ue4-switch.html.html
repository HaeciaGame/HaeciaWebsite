Welcome Haecia players!

By LucioDec 27, 2018

Greetings, everyone! We would like to announce something we have wanted to do for a long time. We hav ewanted to switch from Unreal Engine 4 to Java. Unreal Engine 4 is a great game engine but we felt we needed more customization.

Why did we need to switch?

We needed to switch for many reasons. One great reason is that we felt we were limited by some things. We wanted to be able to customize Haecia and make a great game. We felt like we could not do that with Unreal Engine 4.

Our opinions on Unreal Engine 4

We love Unreal Engine 4 and think it is a great game engine. The switch will not affect the quality of Haecia, but will affect the gameplay. We hope to show you guys in our next release!
